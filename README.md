## 🪟 Install on Windows

After the file finishes downloading:

1. Open File Explorer
2. Go to the Downloads folder
3. Find the file you just downloaded
4. If it is a ZIP file, right-click it and choose Extract All
5. Open the extracted folder
6. If you see an `.exe` file, double-click it
7. If Windows shows a SmartScreen prompt, choose More info, then Run anyway
8. If Windows asks for permission, choose Yes

## 🛠️ If something does not work

If the app does not start:
1. Check that the file finished downloading
2. Extract the ZIP file if needed
3. Right-click the app and choose Run as administrator
4. Make sure Windows is up to date
5. Try the latest release from the download page

If the window opens and closes right away:
1. Open Command Prompt
2. Run the app from the folder
3. Read any error text on the screen
4. Try a fresh download from Releases

If you get a blocked file message:
1. Open the file properties
2. Check for an Unblock option
3. Apply it if present
4. Try opening the app again
